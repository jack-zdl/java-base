## 冒泡排序
```
   4 3 1 7 9 6 2 0
   将第一的坑填入正确的数字，比如正确排序第一个位置应该是0.所以冒泡排序是求第一位置的数字。
   时间复杂度 O(n^2)  空间复杂度 O(1)
   第一排序 以第一的数字为基数，以第二个数字跟基数比较，总是将最小的比较值放入第一坑位。一直轮询到最后一个数字。这样第一个坑位是最小的值，
            第一个坑就不改变。
   第二个排序 以第二个数字为基数，以第三个数字跟基数比较，将最小的值放在第二个坑位。
   以此类推。。。
   当排完最后一个，顺序就是从小到大的顺序。

```
